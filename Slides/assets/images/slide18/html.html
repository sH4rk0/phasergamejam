La prima interazione con un GameObject

Per prima cosa creiamo 2 variabili, una di tipo Phaser.GameObjects.Text ed una di tipo Phaser.GameObjects.Image come fatto in precedenza.

Assegnamo alle due variabili i nostri GameObjects.

this._text = this.add.text(this.game.canvas.width / 2, 450, "0");

this._image = this.add .image(this.game.canvas.width / 2, 550, "sprite") .setScale(2) .setInteractive();

Ecco dove avviene la magia...

this._image.on("pointerup", () => { this._clicks += 1;
this._text.setText("" + this._clicks);
