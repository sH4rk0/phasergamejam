ARCADE PHYSIC

Aggiungiamo la gravità!

Perchè la forza di gravità agisca sui nostri game objects la prima cosa che dobbiamo fare è attivare la Arcade Physic nel nostro file InitGame.ts

physics: {
default: "arcade",
arcade: {
debug: true,
gravity: { y: 1 }
}
}

Poi inseriamo il solito codice per inserire uno sprite nella scena, ma con una piccola modifica...

this.physics.add
.image(640, 400, "sprite")
.setScale(2)
