Desktop and mobile HTML5 game framework

A fast, free and fun open source framework for Canvas and WebGL powered browser game.

Games can be compiled to iOS, Android and native apps by using 3rd party tools. You can use JavaScript or TypeScript for development.

Created by Richard Davey @photonstorm
