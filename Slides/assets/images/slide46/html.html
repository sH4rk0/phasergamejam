Nessun tipo vs alcuni tipi

Void

Il tipo di dato void indica l’assenza di un valore ed è normalmente utilizzato per indicare che una funzione non restituisce alcun valore.

miafunzione():void{
let _numero:number = 3 + 4;
}

miafunzione2():number{
return 3 * 2;
}
