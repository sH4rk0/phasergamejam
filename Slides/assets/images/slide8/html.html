Per caricare un nostro asset e renderlo disponibile alla nostra scena, dobbiamo precaricarlo nel metodo PRELOAD

preload {
this.load.image("logo", "assets/images/logo.png");
}

Fatto questo possiamo inserire nel metodo CREATE il seguente codice...

create {
this.add.image(0, 0, "logo");
}
