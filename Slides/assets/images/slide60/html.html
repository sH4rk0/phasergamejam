Package.json

La lista della spesa

il package.json è un file JSON che si trova nella ROOT di ogni progetto Javascript/Node. Contiene metadati rilevanti per il progetto ed è usato per gestire le dipendenze del progetto, gli script, la versione e molto altro.

{
"name": "phaser-project-template",
"version": "3.55.2",
"description": "Phaser 3 starter template with TypeScript and webpack.",
"homepage": "https://github.com/yandeu/phaser-project-template#readme",
"main": "index.js",
... e molto altro ancora!
