anims.create() - step 2

Animiamo il nostro primo sprite

Adesso creiamo la nostra animazione, o meglio configuriamola creando una variabile locale attraverso l'istruzione let

let _animation = { key: "run", frames: this.anims.generateFrameNumbers("sprite", { frames: this._runAnimation }), frameRate: 10, yoyo: false, repeat: -1 };

Passiamo la nostra variabile al metodo anims.create()

this.anims.create(_animation);

Infine chiediamo al nostro sprite di eseguire l'animazione run

this._sprite.play("run");
